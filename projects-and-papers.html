JSflare
A Dynamic DNS client using Cloudflare

JSflare is a simple command-line tool that keeps your Cloudflare DNS records updated with your current public IP address. Ideal for users with dynamic IPs who want to host services at home, it supports both Cloudflare API keys and tokens for secure authentication. Configure it easily with a JSON file and let it handle the rest.

The source code is available on GitHub.

AmznShipWarn
Know when an item isn't shipped by Amazon

AmznShipWarn is a lightweight browser extension that alerts you whenever an item on Amazon is not shipped directly by Amazon. Designed to help users avoid unexpected shipping costs or third-party sellers, it works across multiple Amazon domains and integrates seamlessly into both Firefox and Chromium-based browsers.

The extension is available on Firefox and Chrome stores.

The source code is available on GitHub.

EBNF Railroad Visualizer
Visualize EBNF grammars as interactive railroad diagrams

EBNF Railroad Diagram Visualizer is a browser-based tool that turns EBNF grammars into interactive railroad diagrams. It supports Wirth Syntax Notation, highlights errors, and makes sharing grammars easy with URL encoding.

The visualizer is hosted on a best-effort basis on this website and on Github Pages.

The source code is available on GitHub.

Eugen
From KUSSS data to organized Discord servers

Eugen is a Kotlin-based Discord bot that automatically manages course channels using KUSSS data. It creates and assigns channels for students, supports simple join/leave commands, and reduces server administration so users can focus on studying (or memes).

The source code is available on GitHub.

System Software Project
First-fit allocator with Mark & Sweep Garbage Collection

A System Software course project implementing a memory manager in C++. Utilizing first-fit allocation and a Mark & Sweep garbage collector based on the Deutsch-Schorr-Waite algorithm.

The source code is available on GitHub.
